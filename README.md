# VK_VLM_Project
Русскоязычная модель, которая понимает изображения. Цель проекта: дообучить модель на открытых данных VK и получить измеримый прирост accuracy. Подробнее в README.


# Подробное описание проекта

## 1. Постановка задачи
Готовые компактные модели не всегда уверенно работают с русскоязычными вопросами и локализованными бенчмарками.
Проект посвящён адаптации визуально-языковой модели к русскоязычным вопросам по изображениям. Идея в том, чтобы получить измеримое улучшение качества на VK MMBench-ru после дообучения на открытом датасете VK.

## 2. Цель (желаемый результат) и задачи
Цель: дообучить компактную VLM и повысить accuracy.

Задачи:
1. Изучить структуру открытых данных VK.
2. Подготовить русскоязычные пары «изображение — вопрос — ответ».
3. Загрузить базовую модель Intel/llava-gemma-2b в 4-битном режиме.
4. Настроить параметро-эффективное обучение LoRA.
5. Измерить accuracy базовой модели.
6. Дообучить модель и повторить оценку на той же тестовой выборке.
7. Сравнить полеченные результаты.

## 3. Данные VK
Для обучения использован deepvk/LLaVA-Instruct-ru. 

Для оценки использован deepvk/MMBench-ru. Модель получала изображение, подсказку, вопрос и варианты ответа. 

## 4. Модель
Базовая модель: Intel/llava-gemma-2b. 

## 5. Дообучение
Использован LoRA-адаптер для проекций внимания q, k, v, o и слоёв MLP языковой части. Параметры: rank 8, alpha 16, dropout 0.05, gradient accumulation 4. Обучалось около 9.8 млн параметров.

Сначала проведён пилот на 20 примерах. Затем выполнено дополнительное обучение на 100 примерах с learning rate 1e-4 в течение одной эпохи. 


## 6. Методика оценки
Для честного сравнения до и после обучения использованы одни и те же 100 примеров. Accuracy вычислялась как доля верный ответов.

## 7. Результаты
- Базовая модель: 42/100, accuracy 42%.
- После LoRA: 52/100, accuracy 52%.
- Прирост: +10%.
- Пилот на 20 примерах: 45% до и 55% после.

## 8. Вывод
Цель достигнута: даже короткое обучение на открытых русскоязычных данных VK улучшило результат модели на выбранной тестовой выборке. Развивая эту идею можно добиться лучшего результата в будущем для русскоязычных пользователей.
